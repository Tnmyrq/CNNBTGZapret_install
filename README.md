# CNNBTGZapret_install

Сейчас на данный момент есть выбор среди методов обхода:
1. Под Windows 10-11
2. Под Android (в разработке)

## Windows10/11 Version:

Инструкция по запуску:
📁Инструкция:
1. Установите данную программу на свой Windows 10/11. В любое удобное место.
2. Запустите её, желательно от имени администратора.
3. Далее у вас откроется меню, здесь вы видите начальную инструкцию, можете её заскринить на будущее.
4. Обязательно ставим галочку в пункте Открыть прокси в Telegram сейчас.
5. Далее нажмите "Начать".
6. У вас откроется телеграм, с предложением подсоединения прокси, нажмите: Включить.

✅Всё, программа инициирована.
Программа работает в системном трее(в фоновом режиме, и никак  не влияет на другие программы.

🗂Так-же можете нажать на кнопку "Показать скрытые значки" в панеле задач, там вы увидите иконку синего кружка с буковкой "Т", нажмите правой кнопкой мыши, и там вы сможете пере подключить прокси в телеграм(первый пункт), или закрыть его(последний пункт). Ещё есть логи и настройки.

## Android версия 1.0 Beta

Данная версия CNNB TG Zapret предствалена для мобильных устройств на базе Android. Приложение находиться в бета-тестировании и может работать не совсем стабильно. Мы активно работаем над выводом из Beta тестирования. Следите за обновлениями ниже, чтобы не пропустить актуальные версии CNNB TG Zapret.

> Для получения доступа к расширенному режиму "Код", Вам будет необходимо ввести код: "devmanage" без скобок.
> В расширенном режие вы сможете настроить конфигурацию подключения.
